FaceRec - Face recognition with OpenCV

	
	

	The application uses some algorithms from:
	Chapter 8 of the book "Mastering OpenCV with Practical Computer Vision Projects", Packt Publishing, 2012.
	http://www.packtpub.com/cool-projects-with-opencv/book



Requirements: 	OpenCV v2.4.1 or later (from June 2012) to compile and run the application.
		Qt 5.0 or later to compile the application.
		At least 3 Face & Eye detection XML files from OpenCV, which you can copy from OpenCV "data" folder to the folder where exe file is.
